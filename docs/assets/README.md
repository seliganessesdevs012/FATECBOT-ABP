# 🖼️ assets — Recursos Visuais da Documentação

> Pasta reservada para imagens, diagramas e outros artefatos visuais usados pela
> documentação do FatecBot.

***

## O que entra aqui

- Logo do projeto
- Diagramas exportados para PNG ou SVG
- Ilustrações usadas em READMEs e documentos de arquitetura

***

## Regras de uso

- Prefira nomes de arquivo descritivos e em kebab-case
- Referencie assets sempre por caminho relativo
- Não use imagens externas em documentos do projeto quando o arquivo puder viver aqui
- Se um diagrama fizer parte de um documento canônico, mencione o arquivo também no texto do documento

***

## Estado atual

Atualmente a pasta contém o logo do FatecBot:

- `fatecbot-logo.png`

Novos diagramas arquiteturais podem ser adicionados aqui conforme forem produzidos.
